C Language


C is a general-purpose programming language created by Dennis Ritchie at Bell Labs in 1972.


It remains popular due to its fundamental role in computer science, providing low-level access to memory and system resources.

C Variables


Variables in C are used to store data that can be modified during program execution. Each variable in C must be declared with a specific data type.

Common data types include int, char, float, and double.


Example:

                    
            int age = 25;  // Integer variable
            char grade = 'A';  // Character variable
            float height = 5.9;  // Floating-point variable

C Functions


Functions in C allow for modular programming by breaking down a program into smaller, manageable blocks of code. A function is defined by its return type, name, and parameters.


Example:

                    
            #include 
            
            int add(int a, int b) {
                return a + b;
            }
            
            int main() {
                int result = add(5, 10);
                printf("Sum: %d\n", result);
                return 0;
            }

C Arrays


Arrays in C are used to store multiple values of the same data type in a contiguous block of memory.

Arrays are accessed by their index, with the first element having an index of 0.


Example:

                    
            int numbers[] = {1, 2, 3, 4, 5};  // Array of integers
            
            printf("First element: %d\n", numbers[0]);  // Accessing first element

C Pointers


Pointers in C store the memory address of another variable. They are powerful tools used for dynamic memory allocation and manipulation of data.


A pointer is declared using the * symbol, and the & operator is used to obtain the address of a variable.


Example:

                    
            #include 
            
            int main() {
                int num = 10;
                int *ptr = #  // Pointer to num
                
                printf("Address of num: %p\n", ptr);
                printf("Value of num: %d\n", *ptr);  // Dereferencing pointer to get value
                
                return 0;
            }

Python


Python is a widely used programming language created by Guido van Rossum in 1991.


It is used for web development, software creation, mathematics, and system scripting.


What are Pointers in Python?


In Python, variables store references to objects in memory rather than direct memory addresses like in C or C++.


Python does not support traditional pointers, but similar behavior is achieved using object references.


Example:

                    
a = [10, 20, 30]
b = a  # 'b' now points to the same object as 'a'

b[0] = 100
print(a)  # Output: [100, 20, 30] - Changes in 'b' reflect in 'a'

Python Functions


Functions are reusable blocks of code that allow you to perform specific tasks. You define a function using the def keyword in Python.


Example:

                    
            def greet(name):
                return f"Hello, {name}!"
            
            print(greet("Zin"))  # Output: Hello, Zin!

Python File Handling

Python allows file operations using functions like open(), read(), and write() to handle files for reading, writing, and manipulating data stored in files.

The open() function is used to open a file, and you can specify the mode (e.g., 'r' for reading, 'w' for writing).


Example:

                    
            # Opening a file to write
            with open("example.txt", "w") as file:
                file.write("Hello, Python!")
            
            # Opening a file to read
            with open("example.txt", "r") as file:
                print(file.read())  
                # Output: Hello, Python!

HTML


HTML (HyperText Markup Language) is the standard language for structuring web pages.


It uses tags to define elements such as headings, paragraphs, images, and links.


A basic HTML document includes essential elements like <html>, <head>, and <body>.

HTML Basics


HTML (HyperText Markup Language) is the standard language used to create and design web pages. It structures the content on the web and defines elements such as headings, paragraphs, links, and images.


A simple HTML document includes the <html>, <head>, and <body> tags. The <head> contains meta information, and the <body> contains the content that appears on the webpage.

HTML Elements


HTML elements are the building blocks of an HTML document. Each element is enclosed by opening and closing tags. Common HTML elements include headings, paragraphs, links, images, and lists.


For example, the <h1> element represents a top-level heading, and the <p> element represents a paragraph.
